# Megatron Bridge Overview

Megatron Bridge connects Hugging Face Transformers and Megatron Core for scalable model conversion, training, fine-tuning, and checkpoint export. It provides architecture-specific bridges, model providers, training recipes, and performance tooling for language, vision-language, audio, and multimodal models.

Megatron Bridge is designed for ML engineers, researchers, and platform teams that need reproducible large-model workflows across single-node development and multi-node training environments.

## Core Building Blocks

| Building Block | Description |
| --- | --- |
| Model bridges | Convert Hugging Face model configs and weights into Megatron-compatible providers and parameter mappings. |
| Training stack | Runs pretraining, SFT, PEFT, checkpointing, logging, and profiling with a PyTorch-native training loop. |
| Model recipes | Provides model-family-specific training and fine-tuning configurations. |
| Model optimization | Integrates ModelOpt workflows for quantization, distillation, and pruning. |
| Supported model catalog | Documents conversion, inference, and training notes for supported model families. |

## Benefits

- Use high-level conversion APIs while preserving model-parallel checkpoint handling.
- Configure scalable training with tensor, pipeline, context, expert, and data parallelism.
- Start from tested recipes for common model families and customize them for production workloads.
- Export trained Megatron checkpoints back to Hugging Face-compatible artifacts for downstream use.

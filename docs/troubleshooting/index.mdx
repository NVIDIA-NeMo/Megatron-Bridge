# Troubleshooting

Use this section to diagnose setup problems, runtime failures, known release issues, and training performance regressions.

## Quick Checks

1. Confirm that the correct container or Python environment is active.
2. Confirm that the expected Hugging Face token is available when using gated models.
3. Confirm that `3rdparty/Megatron-LM` is initialized and on the expected branch.
4. Review model-family notes for required patches, dependencies, and validated parallelism settings.

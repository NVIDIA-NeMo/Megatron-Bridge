# Getting Started

Start here if you are installing Megatron Bridge, running your first conversion workflow, or migrating from another training stack.

## Before You Start

Before you begin, verify the following:

- Access to a Linux environment or NeMo Framework container.
- Python and package tooling compatible with this repository.
- GPU access for realistic conversion, inference, or training workflows.
- Hugging Face access tokens for gated model repositories when required.

# Megatron Bridge

Megatron Bridge is a PyTorch-native library in the NVIDIA NeMo Framework for converting Hugging Face models to Megatron Core, training and fine-tuning large models, and exporting checkpoints back to Hugging Face-compatible formats.

Use this documentation to get started with model conversion, configure distributed training, find model-family guides, and review release compatibility.

### Code Quality Refactors

- **Remove model-specific providers** — rely on bridge directly to create model configs ([PR#2599](https://github.com/NVIDIA-NeMo/Megatron-Bridge/pull/2599), [PR#2235](https://github.com/NVIDIA-NeMo/Megatron-Bridge/pull/2235))
- **Flatten recipes** — combine perf recipes and library recipes into a single surface
- **Decouple model builder from model config**
